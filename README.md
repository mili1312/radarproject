Mini Radar / Object Detection Simulation (MATLAB)

This project implements a simple radar system simulation in MATLAB that demonstrates how a radar can detect a target and estimate its distance in a noisy environment.

The simulation models the basic radar signal processing chain:

Transmission → Target Reflection → Noise Channel → Matched Filter Detection → Range Estimation

The system transmits a Gaussian pulse, receives a delayed and noisy echo from a simulated target, and estimates the target distance using cross-correlation (matched filtering).
